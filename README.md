# AI Mark to Docs

A clean, fast, and elegant single-file web application for previewing and exporting AI-generated Markdown content into professional documents.

Built with a focus on simplicity, high-quality math rendering, and seamless export workflows.

![Preview](https://postimg.cc/5Q2gTpPg))  

## Features

- **Live Markdown Preview** with GitHub Flavored Markdown support (tables, lists, code blocks, etc.)
- **Full KaTeX Math Support** — inline and block equations render perfectly
- **Syntax Highlighting** for code blocks
- **Multiple Export Formats**:
  - DOCX-ready HTML (optimized for Microsoft Word)
  - PDF (via browser print with clean pagination)
  - Self-contained HTML
- **Modern Glassmorphism UI** with light/dark theme support
- **Responsive split-view editor**
- **Completely client-side** — no data leaves your browser

## How to Use

1. **Download** the `md-to-doc.html` file from this repository.
2. **Open** it in any modern web browser (Chrome, Edge, Firefox, Safari).
3. **Paste** your AI-generated Markdown into the editor on the left.
4. **Watch** real-time rendering with proper math equations on the right.
5. **Export** using the buttons:
   - **DOCX**: Download an HTML file optimized for Word → open in Microsoft Word → Save As `.docx`
   - **PDF**: Opens print dialog (choose "Save as PDF")
   - **HTML**: Downloads a standalone HTML file
   - **Copy Markdown**: Quick copy to clipboard

**Keyboard Shortcut**: `Ctrl + S` to trigger DOCX export.

## Tech Stack

- **Frontend**: Vanilla HTML, CSS (Tailwind CSS), JavaScript
- **Markdown**: [Marked.js](https://github.com/markedjs/marked)
- **Math Rendering**: [KaTeX](https://katex.org)
- **Code Highlighting**: [Highlight.js](https://highlightjs.org)
- **Styling**: Tailwind CSS + custom glassmorphism
